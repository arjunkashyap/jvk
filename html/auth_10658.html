ओंकारेश्वरानंद, स्वामी त्यांचे लेख

  • स्वामी प्रेमानंदांच्या सहवासात-१ केवळ त्यागानेच परम शांती (मार्च १९८६)
  • स्वामी प्रेमानंदांच्या सहवासात-२ धर्मसमन्वय व श्रीरामकृष्ण (एप्रिल १९८६)
  • स्वामी प्रेमानंदांच्या सहवासात-३ कर्मयोग (मे १९८६)
  • स्वामी प्रेमानंदांच्या सहवासात-४ अहंकारच माया (जून १९८६)
  • स्वामी प्रेमानंदांच्या सहवासात-५ श्रीरामकृष्ण : सर्वच संप्रदायांसाठी आदर्श (जुलै १९८६)
  • स्वामी प्रेमानंदांच्या सहवासात-६ पौर्वात्य व पाश्चात्य सभ्यता (ऑगस्ट १९८६)
  • स्वामी प्रेमानंदांच्या सहवासात-७ एकमात्र परमेश्वरच 'आपला' आहे (सप्टेंबर १९८६)
  • स्वामी प्रेमानंदांच्या सहवासात-८ रामकृष्ण मिशनचा उद्देश (ऑक्टोबर १९८६)
  • स्वामी प्रेमानंदांच्या सहवासात-९ श्रीरामकृष्ण-सजीव उपनिषद (डिसेंबर १९८६)
  • स्वामी प्रेमानंदांच्या सहवासात-१० जगात जागृती कोण घडवू शकेल? (जानेवारी १९८७)
  • स्वामी प्रेमानंदांच्या सहवासात-११ बिदगाँ आणि कलमा येथे स्वामी प्रेमानंद (फेब्रुवारी १९८७)
  • स्वामी प्रेमानंदांच्या सहवासात-१२ नियमपालन हीच धर्माची धुळाक्षरे (फेब्रुवारी १९८७)
  • स्वामी प्रेमानंदांच्या सहवासात-१३ श्रीरामकृष्ण : सर्व धर्मांसाठी आदर्श (एप्रिल १९८७)
  • स्वामी प्रेमानंदांच्या सहवासात-१४ समन्वयमहाचार्य श्रीरामकृष्ण (मे १९८७)
  • स्वामी प्रेमानंदांच्या सहवासात-१५ पीर प्रेमानंद (जून १९८७)
  • स्वामी प्रेमानंदांच्या सहवासात-१६ बेलुर मठातील सत्संग (जुलै १९८७)
  • स्वामी प्रेमानंदांच्या सहवासात-१७ जीवन घडवा (ऑगस्ट १९८७)
  • स्वामी प्रेमानंदांच्या सहवासात-१८ माणूस घडविणारे विद्यापीठ (ऑक्टोबर १९८७)
  • स्वामी प्रेमानंदांच्या सहवासात-१९ साधन-समर (नोव्हेंबर १९८७)
  • स्वामी प्रेमानंदांच्या सहवासात-२० सत्संग (डिसेंबर १९८७)
  • माताजींच्या आठवणी (ऑक्टोबर १९९७)
  • स्वामी विज्ञानानंदांच्या आठवणी (नोव्हेंबर १९९७)
  • माताजींच्या आठवणी (जानेवारी १९९८)
  • माताजींच्या गोष्टी (मे १९९८)
  • श्रीरामकृष्ण पार्षदांच्या पुण्यस्मृती (सप्टेंबर १९९८)
  • श्रीरामकृष्ण पार्षदांच्या पुण्यस्मृती (ऑक्टोबर १९९८)
॥श्री रामकृष्णार्पणमस्तु॥
